1.使用命令建立项目
2.使用页面建立项目


----------------------------------------------------------------------------------------------------------

1.使用命令建立项目
vue cli 3
1.安装 cnpm
npm install -g cnpm --registry=https://registry.npm.taobao.org

2.安装 cli:
cnpm install -g @vue/cli

3.创建项目:
vue create hello_world

4.启动服务:
npm run serve



--------------------------------------------------------

2.使用页面建立项目

vue ui
运行之后，就可以在页面进行项目的创建

http://localhost:8000/dashboard






------------------------------------------------------

Vue Cli:为单页面应用 (SPA) 快速搭建繁杂的脚手架。


安装 NodeJs之后，cmd执行
vue init webpack todolist


运行项目,cmd执行
npm run dev
